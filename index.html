Introduction

JoyAI-VL-Interaction focuses on three core interaction capabilities: autonomous interaction, real-time response, and long-term memory. The model continuously observes live visual information, understands the user's goal, chooses when to respond, reacts quickly when the current scene requires attention, and recalls earlier details during interaction.

The following cases compare JoyAI-VL-Interaction with common multimodal assistants in continuous visual tasks. The key distinction is whether the model can keep tracking the changing scene, initiate useful responses at suitable moments, achieve sub-second responsiveness when an event happens, and answer with details from earlier context.

Core Case Studies

These cases are organized around three abilities required by live multimodal agents: autonomous interaction, real-time response, and long-term memory.

Core Ability 01

Autonomous Interaction

The model combines real-time visual information with the user's instruction, then autonomously responds at appropriate moments like a human companion. This ability applies to video calls, mobile shopping companionship, outfit feedback, operation guidance, and sports commentary.

Demo 01

JD App Shopping Companion

Mobile shopping

While browsing the JD app, the user asks the model to help select shoes under a budget constraint. A useful assistant should not stop at the frame where the instruction is issued; it should keep observing the app screen, follow product changes, and naturally continue the conversation.

User instruction: “看看哪个鞋子适合我，预算400元以内”

JoyAI-VL-Interaction

Combines the live app screen with the user's shopping goal and proactively carries out multi-turn interaction as new shoes appear.

Doubao

Responds only after the user's explicit question and keeps describing the shoes visible at the moment the instruction was issued.

Gemini

Responds only after the user's explicit question and remains focused on the product state from the initial query moment.

Demo 02

Pet Livestream Emotion Narration

Live companion

In a pet livestream, the model is asked to describe the pet's emotional state in a personified style. The challenge is continuous responsiveness: the assistant should react as different pets and behaviors appear, instead of answering once and going silent.

User instruction: “实时拟人化描述宠物的情绪状态”

JoyAI-VL-Interaction

Generates personified emotional descriptions in real time according to the pet's changing actions and visible emotional cues.

Doubao

Describes only three pet emotions in the first 13 shots, then gives no further responses for the remaining pets.

Gemini

Describes only the pet emotion at the moment the instruction is issued, then gives no further responses for the remaining pets.

Core Ability 02

Real-Time Response

The model observes and understands what is happening right now, then responds with sub-second latency when the situation calls for action. This ability applies to alerts, home security, factory monitoring, elderly and child care, livestreams, and sports commentary.

Demo 03

Fire Monitoring Alert

Security alert

In a monitoring scenario, the user asks the model to alert immediately when a fire starts. The key requirement is not only recognizing the event, but responding at the moment it happens without hallucinating before the event appears.

User instruction: “当火灾发生时立刻提醒我”

JoyAI-VL-Interaction

Sends an alert within one second after the fire occurs, matching the timing required for practical monitoring and warning scenarios.

Doubao

Sends the alert roughly three seconds after the fire occurs, showing a noticeable delay for time-sensitive warning tasks.

Gemini

Immediately claims that a fire appears after the instruction is issued, but gives no response when the fire actually occurs.

Demo 04

Supermarket Product Detection

Instant reminder

While browsing a JD convenience-store scene, the user asks to be reminded as soon as a specific milk product appears. This case tests whether the model can stay grounded in the live scene and react immediately to a target visual cue.

User instruction: “看到悦鲜活鲜牛奶时立刻提醒我”

JoyAI-VL-Interaction

Sends the reminder the moment the Yuexianhuo fresh milk appears in the visual stream.

Doubao

Sends the reminder about five to six seconds after the target product has already appeared.

Gemini

Responds much later and describes only the scene from the query moment, saying it saw dried fruit products rather than the target milk.

Core Ability 03

Long-Term Memory

The model remembers previous events and fine-grained details like a human observer, with context that can last up to around 12 hours. During interaction, it can combine the current instruction with earlier visual memories. This ability applies to personalized assistants, 24/7 video companions, and security monitoring.

Demo 05

Convenience-Store Price Recall

Long memory

While browsing a JD convenience-store scene, an item briefly appears early in the video. The user asks about it much later, testing whether the model can recall a fleeting visual detail rather than relying only on the current frame.

User instruction: “回忆之前一闪而过的物品标价”

JoyAI-VL-Interaction

Jinsangzi appears at 2s. When asked at 17s, the model accurately recalls the price of Jinsangzi Houbao from earlier context.

Doubao

Answers with the price of a different product, Renhe Runshuang Jinyinhua at 16 yuan, instead of the briefly shown Jinsangzi Houbao.

Gemini

Says it will help look for the item, appears to dispatch an external tool, and after about one minute replies that it did not see Jinsangzi Houbao.
